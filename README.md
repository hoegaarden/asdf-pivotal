# asdf-github-tools
ASDF plugin to install tools from github

## Supported Tools
- antctl
- bosh
- credhub
- fly
- govc
- imgpkg
- kubelogin
- om
- pivnet
- sonobuoy
- s5cmd
  
## Installation
To manually install use:
```
asdf plugin-add <tool name> https://github.com/laidbackware/asdf-github-tools.git
```
Soon to be built into ASDF.
